Payload pix com chave Copia e Cola

Uma payload simples que gera um QRcode de pagamento via pix. Os valores de entrada para geração do QRcode são: nome, cidade, valor, chave pix e o identificador. Sendo obrigatórios todos os valores com exceção do identificador. O programa é modulazirado em três partes, sendo: main.py (responsável pela coleta e verefocação dos dados e exibição na GUI), link.py (responsável por criar o link para pagamento com os dados coletados e vereficados, a chave pix já funciona aqui) e genQrCode (responsável por gerar o QRcode com o link gerado pelo link.py). Há poucas funcionalidade para adicionar futuramente.

Validador de catões

Programa único que recebe o número do cartão bancário via terminal e retorna se o cortão é matematicamente válido. Ele retorna também a bandeira do cartão baseado na composição e a quantidades de caracteres do número fornecido, sendo capaz de identificar as seguites bandeiras: Visa, Mastercard, American Express, Discover e Diners Club. O código conta com uma vereficação robusta de tipo e quantidade de caracteres. Pretendo colocar um GUI no projeto futuramente.

Gerador de CPFs

Um gerador de CPFs válidos matematicamente, feito em Python. Este projeto gera números aleatórios e calcula os dígitos verificadores corretamente, sem depender de CPFs reais. Ideal para testes, simulações e aprendizado. Particularmente, foi o projeto que mais gostei de fazer. Ele foi muito desafiador! Futuramente quero usar esse projeto como base para outro, que ira juntar este projeto e o validador de CPFs.

Vereficador de CPFs

Programa modular em duas partes, sendo: functions.py (parte responsável por relizar os calculos de vereficação do CPF, e somente a vereficação se ele é válido) e main.py (responsável por criar o GUI e filtrar somente o dado válido para vereficação). Já considero o projeto finalizado.

Gerador de senhas aleatórias

Um programa simples que gera senhas aleatórias, podendo personalizar vários aspectos da senha gerada. Aqui eu uso a própria biblioteca random do python, portanto caso queira usa-lo, basta ter o python instalado no dispositivo. Pretendo futuramente colocar um GUI no projeto. O documento está muito bem documentado.

Cotações com GUI

É um programa modular com duas partes, sendo: get.py (responsável por fazer uma requisição a uma API pública, retornar os dados requeridos e processar eles) e o app.py (responsável por pegar os dados do get.py e exibir no GUI). Futuramente quero retomar o projeto e colocar mais opções de moedas, tanto na compra quanto na venda, pois por enquanto ele só funciona com BRL ⇛ DLR ou EUR.
